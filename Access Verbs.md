## Access Verbs

| Verb           | desc                           |
| -------------- | ------------------------------ |
| auth           | 认证                           |
| read_self      | 允许读取自己的用户信息         |
| read_internal  | 允许读取小组成员的用户信息     |
| read_other     | 允许读取其他小组成员的用户信息 |
| write_self     | 允许修改自己的用户信息         |
| write_internal | 允许增删小组成员或修改信息     |
| write_other    | 允许增删其他小组成员或修改信息 |
| read_group     | 允许读取小组信息               |
| write_group    | 允许修改小组信息               |
| alter_group    | 允许增删小组                   |
